Doorman
=======
Repo for a front-door automation project.


Hardware
--------------
Aiphone VC-K



Power:
----------
12V DC at some low current

Terminals:   [1][2][3][4][B]
-----------------------------------
1-Audio out  
2-Audio in  
3-GND  
4-Door Release (connect to GND to unlock door)  
B-Buzz	(0V = standby, 12V = buzzer being pressed at entrance)
